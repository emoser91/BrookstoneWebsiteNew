Platform Requirements

Acceptable files and job requirements for Brookstone Printing:

  • Quark (12.2.3.0 or lower)
  • InDesign (CC)
  • Adobe Illustrator/Vector files (CC)
  • Press-Quality PDF with a minimum of .125” bleeds (if it includes bleeds). CMYK, fonts outlined or embedded, 300dpi or higher for any raster images.
  • PSD, TIF or JPEG are acceptable, but we do not guarantee any text/copy created in these formats will print clearly at print-resolution. Please use 1200dpi for any bitmaps to ensure highest quality output.
  • For any Microsoft Office files (Word, Publisher, Etc.), please convert to a CMYK, Press-Quality PDF with bleeds (if applicable) and send the native file along for reference.

Artwork Requirments

Artwork sent to Brookstone Printing needs to conform to the following:

  • All fonts must be outlined, embedded or included
  • All links must be embedded or included
  • All images must print at 300dpi or higher
  • CMYK, Spot or Grayscale only - No RGB
  • A minimum of .125” bleed must be included on any job that requires a bleed
  • Native files must be packaged (to include fonts and links)
  • A reference PDF must be included with all packaged native files

* Additional fees may apply if supplied art must be converted or altered beyond the previously outlined job/quote specifications.

File Transfer Procedure

Brookstone Printing procedure for FTP file transfer:

Brookstone Printing has FTP and Dropbox accounts for easy file sharing. To send files via FTP, you must have FTP client software such as CyberDuck, FileZilla or Transmit. Additionally, you will need our FTP Settings (shown below) in order to connect to our server. After uploading, please send an email to prepress@badgerprinting.com with the file name and Company name after the upload is complete.

Contact the art department if you need any additional assistance in the file transfer process.

Once the art/prepress department receives the file(s) a preflight will be performed and you will be contacted if there are any items that need to be changed before proofing or printing.
